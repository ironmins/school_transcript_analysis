📋 NEIS 자료 업로드 방법:

성적조회/통계 → 학기말성적조회 → 학기말성적종합일람표 → XLS data 파일을 다운로드하세요

⚠️ 주의: XLS 아닙니다. XLS data 양식을 업로드 해야합니다.

🔒 보안 안내

  • 업로드한 성적 파일은 서버에 전송/저장되지 않습니다.
  • 모든 분석은 사용자의 브라우저(로컬)에서만 수행됩니다.
  • 페이지를 닫거나 새로고침하면 메모리에 남은 데이터는 사라집니다.

즉, 업로드한 양식은 외부로 유출되지 않습니다 🙂
